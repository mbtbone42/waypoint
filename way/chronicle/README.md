# Chronicle

The Chronicle is Waypoint's project memory — a narrative event log of decisions, learnings, retrospectives, pivots, and amendments.

## How to Use

- One file per month: `YYYY-MM.md`
- Entries are chronological within each file
- Each entry has a timestamp, type, author, and links to related artifacts
- Entry types: Decision, Learning, Retrospective, Pivot, Amendment

## When to Add an Entry

- When a significant architectural or design decision is made
- When something is learned through implementation that changes understanding
- When direction changes (pivot)
- When an existing PRD, ADR, or DDD is updated (amendment)
- During retrospectives
